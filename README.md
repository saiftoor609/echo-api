# Echo API

## Project Description
This is a simple Node.js + Express API that echoes back the message sent in a POST request.  
It validates the message using `express-validator` and returns the echoed message in JSON format.

## Packages Used
- express
- express-validator
- nodemon (development)
- jest (testing)
- supertest (testing)
